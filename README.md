# safi-vape-ecommerce
A collaborative academic project building a fully-featured age-restricted e-commerce website for premium vaping products, built with modern web technologies and comprehensive user experience design.

##  Key Features
- Age verification system with regulatory compliance
- Complete shopping cart with localStorage persistence
- Multi-page product catalog with search functionality
- Full checkout process with customer information forms
- Google Apps Script integration for order processing
- Responsive design optimized for all devices

##  Tech Stack
HTML5 · CSS3 · JavaScript · Tailwind CSS · LocalStorage API · Google Apps Script

##  Project Structure
- Homepage with hero section and featured products
- Product catalog with search and filtering
- Complete checkout system with order confirmation
- Contact forms with validation and success feedback
- Legal and policy pages for compliance

##  Advanced Features
- Real-time cart updates across all pages
- Mobile-optimized navigation and UI
- Form validation and user feedback systems
- Professional modal and popup systems
